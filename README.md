# 🛒 Good-Sales_Dashboard

A modern, interactive **Sales Dashboard** built using Streamlit to analyze and visualize sales data.

---

## 🚀 Live Demo

👉 **View Deployed App:**
🔗 https://good-sales-dashboard.streamlit.app/



---

## ✨ Features

* 📊 Interactive dashboard UI
* 📁 Upload your own CSV
* 🔍 Advanced filters (Region, Category, Date)
* 🔎 Smart search
* 💰 KPIs (Sales, Profit, Orders)
* 📈 Sales trends
* 🌍 Regional insights
* 📥 Download data (CSV & Excel)

---

## ⚙️ Installation

```bash
git clone https://github.com/bhavintak143/Good-sales-dashboard.git
cd Good-sales-dashboard
pip install -r requirements.txt
streamlit run app.py
```

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Matplotlib
* Seaborn
* Plotly

---

## 📊 Dataset

Required columns:

* Date
* Region
* Category
* Product
* CustomerName
* TotalSale
* Profit (optional)

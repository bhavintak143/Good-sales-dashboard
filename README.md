# 🛒 Good-Sales_Dashboard

A modern, interactive **Sales Dashboard** built using **Streamlit** to analyze and visualize sales data efficiently. This project provides real-time insights into sales performance, categories, regions, and trends with a clean and user-friendly interface.

---

## 🚀 Features

* 📊 **Interactive Dashboard UI**
* 📁 Upload your own CSV dataset
* 🔍 Advanced filtering (Region, Category, Date)
* 🔎 Smart search (Product, Customer, Region)
* 💰 Key performance indicators (Sales, Profit, Orders)
* 📈 Sales trend analysis (daily & monthly)
* 🧺 Category-wise sales breakdown
* 🌍 Regional sales distribution (Pie chart)
* 🗺️ Region-wise comparison charts
* 📋 Interactive data table view
* 📥 Export filtered data (CSV & Excel)

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/bhavintak143/Good-sales-dashboard.git
cd Good-sales-dashboard
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
streamlit run app.py
```

---

## 📊 Dataset Requirements

Your CSV file should contain columns like:

* `Date`
* `Region`
* `Category`
* `Product`
* `CustomerName`
* `TotalSale`

If `Profit` is not present, it will be automatically calculated.

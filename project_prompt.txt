🛒 GOODS SALES DASHBOARD - PROJECT PROMPT & DOCUMENTATION
================================================================

📋 PROJECT OVERVIEW
-------------------
This is a comprehensive Goods & Sales Dashboard built using Python Streamlit that provides interactive data visualization and analysis for sales data. The project includes data generation, multiple dashboard pages, and various analytical views.

🎯 PROJECT OBJECTIVES
--------------------
1. Create an interactive sales dashboard for goods and products
2. Provide multiple analytical views (EDA, Category Insights, Regional Analysis)
3. Enable data filtering and search functionality
4. Generate visual reports with charts and graphs
5. Allow data export in multiple formats (CSV, Excel)

🔧 TECHNICAL STACK
------------------
• Frontend: Streamlit (Python web framework)
• Data Processing: Pandas, NumPy
• Visualization: Matplotlib, Seaborn, Plotly
• Data Generation: Faker library
• Export: XlsxWriter for Excel files

📈 DATA STRUCTURE
-----------------
The sales dataset contains the following fields:
• OrderID: Unique order identifier (ORD1000+)
• Date: Order date (last 1 year)
• CustomerName: Generated customer names
• Region: North, South, East, West
• Product: Specific products within categories
• Category: Electronics, Clothing, Home Decor, Groceries
• Quantity: Number of items (1-10)
• Price: Product price (₹100-₹50,000)
• TotalSale: Calculated total (Price × Quantity)
• Profit: 20% of total sale

🔧 INSTALLATION & SETUP
-----------------------
1. Install Python dependencies:
   pip install -r requirements.txt

2. Run the main dashboard:
   streamlit run dashboard.py

3. Access individual pages:
   streamlit run pages/EDA.py
   streamlit run pages/category_insights.py
   streamlit run pages/Raw_data.py
   streamlit run pages/Region_heatmap.py
